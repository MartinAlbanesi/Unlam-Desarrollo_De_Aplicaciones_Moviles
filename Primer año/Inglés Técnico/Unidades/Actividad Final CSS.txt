7. Responder
a. CSS es un lenguaje de estilo que define el diseño de documentos HTML.
b. CSS es compatible con todos los navegadores en la actualidad.
c. Los diseñadores comenzaron a buscar posibilidades para agregar diseño a documentos en línea. 
d. CSS se inventó para proporcional oportunidades de diseño sofisticadas compatibles con todos los navegadores para diseñadores web.
e. Los beneficios de CSS son: controlar el diseño de muchos documentos desde una sola hoja de estilo, control más preciso del diseño, aplicar diferentes diseños a diferentes tipos de medios (pantalla, impresión, etc), numerosas técnicas avanzadas y sofisticadas.
